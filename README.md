# Advent of Code 2020

This repo contains my solutions for AdventOfCode 2020.

I used https://github.com/caderek/aoc-starter-ts as a template to automate the tedious tasks of running my solutions.
## Installation


Using NPM:

```
npm i
```

## Running dev mode

```
npm start day<N>
```

Example

```
npm start day5
```
